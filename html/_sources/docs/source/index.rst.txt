
Welcome to ngehtsim's documentation!
====================================

ngehtsim is a set of Python_ tools for generating synthetic data appropriate for the next-generation Event Horizon Telescope (ngEHT_).  ngehtsim builds on synthetic data generation capabilities contained in the ehtim_ library, primarily by adding utilies for incorporating local weather effects, telescope properties, and fringe-finding proxies to the generation procedure.

.. _Python: https://www.python.org/
.. _ngEHT: https://www.ngeht.org/
.. _ehtim: https://achael.github.io/eht-imaging/
